# 切霉豆腐小游戏

一个轻量的网页小游戏：切 4×4 霉豆腐 → 入盆 → 加调料 → 摇一摇 → 装罐。主打可爱手作感的交互流程。

## 功能
- 4×4 霉豆腐切块，切到即落盆
- 加调料（辣椒 / 白酒 / 盐）
- 摇一摇动效
- 装罐动画（从盆口倒入罐口）
- 完成弹窗

## 运行方式
**方式一：直接打开**
- 双击打开 `index.html`（或拖到浏览器中）

**方式二：本地服务（推荐）**
```bash
python3 -m http.server 8000
```
浏览器访问：
```
http://localhost:8000
```

## 控制
- 鼠标 / 触控拖动：切块
- 按钮：选择调料、摇一摇、装罐
